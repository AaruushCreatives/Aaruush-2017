Autodesk Revit Structure

To take full advantage of Building Information Modeling, the Autodesk Revit Structure workshop has been designed to teach the concepts and principles from building design through construction documentation using the Autodesk Revit Structure software. This workshop is intended to introduce Participants to the user interface and the basic building components of the software that makes it a powerful and flexible structural modeling tool.The goal is to familiarize the participants with the tools necessary to create, modify, and document the parametric model.

Virtual Reality Gaming

A workshop creatively designed for gaming enthusiasts, it provides an opportunity for gamers to create and develop their own Virtual 3-dimensional (3D) games. Right from scratch, the workshop indulges its participants to learn how to make 3D objects, create new models, place blue prints, texture and shape animated objects, etc. This entire process is curated using the Maya Software, and the developed 3D game is debugged into a virtual game using the Worldviz Vizard Toolkit. This two-day workshop is concluded by installing these games into smartphones, and played thereafter.

Team: Individual participant, with a complementary Google cardboard VR headset to take away.
Date: 28th- 29th Sept 2016
